(d) Stop: GPIO_WriteBit(PB7, RESET), Delay_Us(5), GPIO_WriteBit(PB6, SET), Delay_Us(5), GPIO_WriteBit(PB7, SET). (e) Re-init: call i2c_util_init() to restore PB6/PB7 as AF_OD and re-init I2C1 peripheral. Return I2C_OK if bus released, else I2C_BUS_FAULT.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCU <-> I2C1 Bus | SCL/SDA exposed to external devices; any slave can hold SDA low |
| I2C1 Bus <-> INA226 | I2C data integrity across PCB traces; noise/glitch susceptible |
| MCU <-> USART1 | Debug output only, no security boundary |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing | I2C1 bus | accept | I2C is a board-level bus with physically fixed devices; no authentication mechanism available or needed |
| T-01-02 | Tampering | I2C1 data lines | mitigate | i2c_status_t return codes checked at every operation; read data is from acknowledged transfers only (NACK detection in i2c_util) |
| T-01-03 | Denial of Service | I2C1 bus (slave stuck) | mitigate | i2c_util_bus_recovery() with 9-clock-pulse GPIO recovery per I2C-03; tick-count timeout on every I2C_CheckEvent poll prevents infinite hang (per D-06, D-07) |
| T-01-04 | Denial of Service | Stack overflow | mitigate | All local arrays bounded (2-byte register read buffers max); no recursion; 32K SRAM monitored — this plan adds ~200 bytes of new state (device structs + stack frames) |
| T-01-05 | Elevation of Privilege | N/A | accept | No privilege model in bare-metal embedded firmware |
| T-01-06 | Information Disclosure | USART1 debug output | accept | Debug output is physically accessible only via PA9 pin; no sensitive data transmitted |
| T-01-07 | Denial of Service | SysTick counter | mitigate | I2C timeout uses 24-bit SysTick->CNT comparison with modulo-safe arithmetic; timeout threshold is configurable I2C_TIMEOUT_MS |
| T-01-08 | Tampering | INA226 calibration | monitor | Calibration register written once at init; Phase 2 will add periodic re-write (PROT-04). For Phase 1, calibration corruption would produce wrong readings but not unsafe operation |
| T-01-SC | Tampering | Package installs | accept | No external packages installed — all code is from existing SPL V2.9 tree. Package Legitimacy Audit in RESEARCH.md confirms N/A. |
</threat_model>
