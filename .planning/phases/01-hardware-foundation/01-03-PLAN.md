## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| MCU <-> USB Host (PC) | USB cable carries enumeration data, CDC ACM protocol — untrusted host |
| MCU <-> USBFS Peripheral | Internal MCU bus — no boundary risk |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Spoofing | USB host enumeration | accept | No device authentication — CDC ACM is a debug interface with no security requirements. VID/PID are WCH development defaults. |
| T-03-02 | Tampering | USB descriptors | mitigate | Descriptors are compile-time const arrays (const uint8_t MyDevDescr[]) — cannot be modified at runtime. Any corruption would cause build failure |
| T-03-03 | Denial of Service | USB-CDC buffer overflow from rapid printf | mitigate | usb_printf() bounds output to USB_CDC_BUF_SIZE (128 bytes) via vsnprintf. Checks USBFS_Endp_Busy[DEF_UEP3] before sending — drops data if buffer busy instead of blocking. Per RESEARCH.md Pitfall 3 mitigation. |
| T-03-04 | Denial of Service | USB disconnect during transfer | mitigate | USBFS_IRQHandler handles SUSPEND and RESET interrupts. usb_printf returns -1 when Endp_Busy — super-loop continues without USB output. No blocking on USB state. |
| T-03-05 | Denial of Service | USBFS clock misconfiguration | mitigate | USBFS_RCC_Init() selects PLLCLK_Div2 when SystemCoreClock==96000000 (verified in existing main.c output). If clock is wrong, USB enumeration fails at host side — no crash on MCU, just no COM port. |
| T-03-06 | Denial of Service | Stack overflow from vsnprintf | mitigate | USB_CDC_BUF_SIZE=128 bytes on stack — within 2KB stack budget. vsnprintf truncates to buffer size. No dynamic allocation. |

| T-03-08 | Elevation of Privilege | CDC_SET_LINE_CODING | mitigate | SET_LINE_CODING handler parses 7 bytes but does not reconfigure any UART — parameters are accepted silently for host COM port compatibility but not acted upon. No reconfiguration surface. |
